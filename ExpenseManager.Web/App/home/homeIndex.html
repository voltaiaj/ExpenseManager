Welcome to Glovia Access Request

Glovia is a Tier-1 financial application, which falls under SOX Governance. All requests for access must be documented and approved by the business unit controller. The users are able to add/change/remove access using the following request forms:

Password Reset/Re-activate Account

Request to reset user password or re-activate user account when not logged in for over 90 days.

Terminate All Access

Request when user no longer requires Glovia access or is no long with the Company. All access will be removed from Glovia.

Network ID Changed

Request for Glovia Login ID change due to Network ID change.

New/Modify Access

Request for New User access to Glovia or Functionality add/change for an existing user.

SOD/SOX Compliance Audit

For audit use only!
